Proof Designer Help

Proof Designer is a website that writes outlines of proofs in elementary set theory, under the guidance of the user. It is intended to help students learn to write proofs. Proof Designer uses JavaScript; if you have trouble getting it to work, make sure that JavaScript is enabled in your browser.

The documents listed below will help you learn how Proof Designer works.

  • Example. This document will lead you through an example that illustrates how Proof Designer works.
  • A Guide to Proof Strategies. For each strategy, this document also explains what command to give in Proof Designer if you want to use that strategy in a proof. For a more complete explanation of these strategies, see my book How To Prove It, published by Cambridge University Press.
  • Complete instructions for how to use all the commands in Proof Designer.
  • Typing expressions in Proof Designer.
  • Suggested problems to do in Proof Designer.
